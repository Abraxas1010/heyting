Lean Anchors & Notes

Round-Trip Contracts

HeytingLean/Contracts/RoundTrip.lean defines the common interface every bridge implements: encode/decode, compatibility proofs, and transport to `Ω_R`.

Tensor Bridge

HeytingLean/Bridges/Tensor.lean uses finite tuples with coordinate interiorisation; see the dedicated card for roadmap and TODO lemmas.

Graph Bridge

HeytingLean/Bridges/Graph.lean reuses the ambient type as the carrier and lifts adjacency from the re-entry order.

Clifford Bridge

HeytingLean/Bridges/Clifford.lean models bivectors and geometric projectors, collapsing them through the Heyting core.

Future Bridges

Geometry/topology, categorical, and probabilistic bridges reference the longer-term plans in `HeytingLean/TBD/`. Clone this template once those start taking shape.

Each green node links to a more detailed bridge card. When new bridges land, duplicate either the Tensor/Graph/Clifford page and update the links here.
