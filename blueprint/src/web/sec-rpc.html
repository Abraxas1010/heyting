9.3 RPC Interface

Definition 9.5 RPC Method

HeytingLean.ProofWidgets.LoFViz.Rpcapply def:state, def:kernel-data

The apply RPC method:

  1. Receives an event from the widget

  2. Updates scene state via stepper

  3. Rebuilds kernel data from journal

  4. Routes to appropriate renderer

  5. Returns BridgeResult with SVG + proof certificates
